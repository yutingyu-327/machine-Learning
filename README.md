# machine-Learning
乳腺癌数据集的机器学习分析项目
1. 数据加载与预处理
使用 ucimlrepo 库从 UCI 机器学习仓库中获取乳腺癌数据集（ID=17）。

将标签（良性/恶性）编码为数值（0 和 1）。

使用 StandardScaler 对特征进行标准化处理（均值为0，方差为1）。

2. 高斯混合模型（GMM）聚类分析
使用 GaussianMixture 对标准化后的数据进行聚类，假设有两个成分（对应良性和恶性）。

预测每个样本属于哪个聚类，并输出误分样本（聚类结果与实际标签不一致的样本）。

输出误分样本的统计信息。

3. 逻辑回归分类
将数据集拆分为训练集和测试集（70% 训练，30% 测试）。

训练一个逻辑回归分类器，并在测试集上评估准确率。

可视化逻辑回归模型的特征权重，并保存为 feature_importance.png。
